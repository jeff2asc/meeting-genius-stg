Project Accomplishments Report - June 15, 2026

* Corporate admin Archive & Unarchive: Corporate Admins can now only archive and unarchive buildings they own. They can see the archive storage but cannot delete anything (only unarchive).
* Master Admin Archive: Master admin has full Archive & Delete control. Created the archive storage interface where delete actions can be executed.
* Archiving Logic: Archiving a building now automatically archives all building-level templates and dependencies (aside from users).
* Login & Data Fetching Fixes: Fixed the underlying proxy and API routing errors that were preventing meetings, users, buildings, companies, and logins from loading correctly.
* Security & Master Control Data (API Gateway Upgrade):
  - Server-Side Isolation: Migrated all critical database mutations (user creation, company details, reference link deletions) from vulnerable client-side queries to secure backend API routes.
  - Signature & Token Verification: Implemented server-side request verification (isAuthorizedRequest()) checking for a valid system API Key (x-api-key) or active user session Bearer token, immediately blocking unauthorized requests with a 401 Unauthorized status.
  - Admin Client Execution: Replaced client-side queries with createAdminClient() calls executed strictly on the backend, preventing database credentials or elevated privileges from being exposed to the browser.
  - Next.js 15 Parameter Upgrades: Refactored dynamic route parameter handling (e.g. app/api/supabase-proxy/[...path]/route.ts) to be compliant with Next.js 15 async paradigms by awaiting parameter promises.
  - No-Content Proxy Handling: Fixed the HTTP proxy response parser to return null body payloads on 204 No Content and 304 Not Modified statuses, preventing native NextResponse constructor exceptions.
* Transcript Task Extraction: Refactored transcript-based task extractions so the UI matches the manual task creation experience.
* Cleaned Mobile View: Optimized layout grids, borders, and overlays to make the mobile view cleaner.
* Voting Expressions: Custom voting formulas and expressions are now clearer and display proper input configurations.
