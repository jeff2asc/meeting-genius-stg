MEETING GENIUS - INITIAL SERVER SETUP (Mar 13, 2026)
Server: ca.meetinggenius.ca (149.5.247.118)
=====================================================

--- PHASE 1: INFRASTRUCTURE (Run once as root) ---
1. Update System: apt update && apt upgrade -y
2. Install Node.js 20: 
   curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
   apt install -y nodejs git
3. Setup User:
   useradd -m -s /bin/bash pm2
   mkdir -p /opt/meetinggenius/app
   chown -R pm2:pm2 /opt/meetinggenius
4. Install Global Tools:
   npm install -g pm2
   apt install -y nginx certbot python3-certbot-nginx

--- PHASE 2: APP DEPLOYMENT (Run as pm2 user) ---
1. Clone Code:
   su - pm2
   cd /opt/meetinggenius/app
   git clone https://github.com/asccreative/meetinggenius2 .
   # Used GitHub Personal Access Token (PAT) for auth
2. Install Deps: 
   npm install --legacy-peer-deps
3. Set Env: 
   nano .env.local 
   (Added OPENAI_API_KEY and NEXT_PUBLIC_RECAPTCHA_SITE_KEY)
4. Build:
   npm run build
5. Start Process:
   pm2 start npm --name "meeting-genius" -- start -- -p 3000
   pm2 save

--- PHASE 3: NETWORKING (Run as root) ---
1. Nginx Config: Created /etc/nginx/sites-available/meeting-genius
2. Enable Site: 
   ln -s /etc/nginx/sites-available/meeting-genius /etc/nginx/sites-enabled/
   rm /etc/nginx/sites-enabled/default
   systemctl restart nginx
3. SSL Security: 
   certbot --nginx -d ca.meetinggenius.ca
4. Startup Logic:
   pm2 startup systemd -u pm2 --hp /home/pm2
   (Ran the generated sudo command)
   su - pm2 -c "pm2 save"
